# 📖 Описание проекта

Данная работа посвящена разработке интеллектуальной системы анализа игровых действий в **Counter-Strike 2** с использованием современных методов машинного обучения.

Система позволяет:

* 📊 извлекать данные из демо-файлов матчей;
* 🎮 преобразовывать игровые события в последовательности признаков;
* 🤖 обучать Transformer-модель для анализа действий игроков;
* 📈 вычислять показатель тактической оптимальности;
* 🔍 визуализировать результаты через аналитический дашборд.

# 🚀 Быстрый старт

## 1. Клонирование проекта

```bash
git clone <repository_url>
cd cs2_ml_pipeline
```

## 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

## 3. Запуск проекта

### Демонстрационный режим

Не требует наличия демо-файлов.

```bash
python main.py --demo-mode --train --dashboard
```

### Анализ реальной демки

```bash
python main.py --demo path/to/match.dem --train --dashboard
```

### Только ETL-конвейер

```bash
python main.py --demo path/to/match.dem
```

### Пакетное обучение

```bash
python auto_train.py --dir D:\demos --epochs 10 --keep
```

# 🛠 Параметры командной строки

| Параметр      | Значение по умолчанию | Назначение                     |
| ------------- | --------------------- | ------------------------------ |
| `--demo`      | —                     | Путь к файлу демо              |
| `--demo-mode` | False                 | Генерация синтетических данных |
| `--train`     | False                 | Запуск обучения модели         |
| `--dashboard` | False                 | Открытие дашборда              |
| `--epochs`    | 10                    | Количество эпох                |
| `--d-model`   | 256                   | Размер скрытого пространства   |
| `--n-layers`  | 8                     | Число слоев Transformer        |
| `--n-heads`   | 4                     | Число голов внимания           |
| `--seq-len`   | 128                   | Длина последовательности       |
| `--lr`        | 3e-4                  | Скорость обучения              |
| `--target-hz` | 8                     | Частота дискретизации          |
| `--cpu`       | False                 | Использование CPU              |
| `--output`    | output                | Каталог результатов            |
| `--player`    | —                     | Фильтр игрока                  |
| `--seed`      | 42                    | Random Seed                    |

# 📝 Примечания

### Работа с архивированными демо

Для файлов `.dem.zst` может потребоваться установка:

```bash
pip install zstandard
```

### Использование GPU

Для ускорения обучения рекомендуется версия PyTorch с CUDA:

```bash
pip install torch --index-url https://download.pytorch.org/whl/cu118
```

### Тестовый режим

Режим `--demo-mode` позволяет протестировать все компоненты системы без наличия реальных демо-файлов.
